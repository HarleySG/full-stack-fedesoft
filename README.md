# curso nodeJs udemy

## console + multiply app

Esta es una aplicación para generar archivos de tablas de multiplicar.


### Paso 1

```
npm install
```

### Paso 2

Abrir consola y ejecutar los comandos según acción:

#### Listar tablas 
```
node app listar -b 1
```
#### Crear tablas 
```
node app crear -b 1
```
